Ubuntu 安装 Docker

使用 APT 安装

以下命令会添加稳定版本的 Docker CE APT 镜像源，如果需要最新或者测试版本的 Docker CE 请将 stable 改为 edge 或者 test。从 Docker 17.06 开始，edge test 版本的 APT 镜像源也会包含稳定版本的 Docker。

  • 安装必要的一些系统工具
  • 安装 GPG 证书
  • 写入软件源信息

使用脚本自动安装

在测试或开发环境中 Docker 官方为了简化安装流程，提供了一套便捷的安装脚本，Ubuntu 系统上可以使用这套脚本安装：

  1. 主要从get.docker.com上下载get-docker.sh
    1. 如果这个访问不了使用我本人的gitee.com/niziming/general_linux_tools/raw/master/install/get-docker.sh
  2. 将get-docker.sh中的镜像换成阿里云

如果出现错误 get-docker.sh: 2: set: Illegal option -

使用以下语句替换换行符后执行

sed -i 's/\r$//' get-docker.sh

执行这个命令后，脚本就会自动的将一切准备工作做好，并且把 Docker CE 的 Edge 版本安装在系统中。

将用户测试加入 docker 组里面

sudo usermod -aG docker root

systemctl start docker systemctl status docker

用户exit
退出后重新登录

查看是否安装docker

docker version
